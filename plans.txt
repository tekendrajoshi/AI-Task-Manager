1️⃣ Web App Structure

You want three main pages:

Dashboard Page
-Overview of tasks (counts, upcoming deadlines, notifications)
-Option to classify by status (Not Started / Pending / Completed) or priority (High / Medium / Low)

AI Chat Page
-Like your current chat HTML
-Users can interact with the AI Agent to add tasks
-Can also handle task updates

Tasks Page
-Shows all tasks fetched from your remote SQL database (Railway)
-Filter tasks by Status or Priority

Sections:
-Not Started
-Pending
-Completed
--Optional sorting (by DueDate, CreatedAt, Priority)

1️⃣ Setup Node.js Backend
=>You’ll create a small server that connects to your Railway MySQL database and returns tasks as JSON.

Steps:
-Install Node.js (if not installed).
-Create a new folder for your backend project.
-Inside, run:
                npm init -y
                npm install express mysql2 cors

=express → server framework
=mysql2 → database connection
=cors → allow your frontend to fetch data


=====================================================================================
2️⃣ Backend Code (server.js)
const express = require("express");
const mysql = require("mysql2/promise");
const cors = require("cors");

const app = express();
app.use(cors());
app.use(express.json());

// Create connection pool to Railway MySQL
const pool = mysql.createPool({
    host: "YOUR_RAILWAY_HOST",
    user: "YOUR_DB_USER",
    password: "YOUR_DB_PASSWORD",
    database: "YOUR_DB_NAME",
    waitForConnections: true,
    connectionLimit: 10,
    queueLimit: 0
});

// Fetch all tasks
app.get("/tasks", async (req, res) => {
    try {
        const [rows] = await pool.query("SELECT * FROM ai_task_manager");
        res.json(rows);
    } catch (err) {
        res.status(500).json({ error: err.message });
    }
});

// Optional: fetch tasks by status
app.get("/tasks/status/:status", async (req, res) => {
    const status = req.params.status;
    try {
        const [rows] = await pool.query("SELECT * FROM ai_task_manager WHERE Status = ?", [status]);
        res.json(rows);
    } catch (err) {
        res.status(500).json({ error: err.message });
    }
});

const PORT = process.env.PORT || 3000;
app.listen(PORT, () => console.log(`Server running on port ${PORT}`));


✅Replace YOUR_RAILWAY_HOST, YOUR_DB_USER, etc. with your Railway database credentials.


3️⃣ Connect Frontend

In your dashboard page, fetch tasks like this:

async function fetchTasks() {
    try {
        const res = await fetch("http://localhost:3000/tasks"); // or your deployed backend URL
        const tasks = await res.json();

        // Example: separate by status
        const notStarted = tasks.filter(t => t.Status === "Not Started");
        const pending = tasks.filter(t => t.Status === "Pending");
        const completed = tasks.filter(t => t.Status === "Completed");

        console.log({ notStarted, pending, completed });
        // You can now render these arrays into HTML sections
    } catch (err) {
        console.error("Error fetching tasks:", err);
    }
}

fetchTasks();
